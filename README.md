# Selenium Test Automation Framework

## 🔍 Overview
This project demonstrates UI test automation using Selenium WebDriver with TestNG and Page Object Model (POM).

## 🧪 Test Scenarios
- Valid Login Test
- Invalid Login Test
- Add to Cart Test

## 🛠 Tools & Technologies
- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager

## 🧪 Test Coverage
- Login functionality
- Invalid login validation
- Add to cart flow

## 🐞 Bugs Identified
- Incorrect cart locator caused navigation failure (Fixed)

## 📄 Test Documentation
- Test cases included
- Bug reports included
  
## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ
3. Run testng.xml
   

## 📁 Project Structure
- pages → Page Object classes
- tests → Test cases
- base → Setup and teardown
